Zero added noise. The existing mu_noisy column already includes realistic ABS sensor noise from the MATLAB simulation. The classifier achieves perfect F1 on the prepared dataset — this is the baseline.
Why F1 macro? Our 3 surfaces are evaluated with equal batch counts (10 batches × 3 surfaces), so the dataset is balanced at evaluation time. In this regime, F1 macro = Balanced Accuracy — both are appropriate. F1 is preferred in ML literature; BAC is more interpretable to automotive/safety engineers. Neither inflates scores due to class imbalance. MCC would also be valid but is less standard in this domain.
